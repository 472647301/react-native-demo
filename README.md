
### 本地运行

1.克隆项目 
```
git clone https://github.com/472647301/react-native-inssues.git
```
2.安装依赖 
```
cd react-native-inssues

yarn install
```
3.启动项目
- iOS: Xcode打开项目ios目录MyApp.xcworkspace文件运行即可
- Android: Android Studio打开项目android目录
